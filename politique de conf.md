# 2. DONNÉES COLLECTÉES

## 2.1. Données collectées auprès de l’Utilisateur

* Identité : nom, prénom, email, numéro de téléphone (optionnel).
* Informations professionnelles : diplôme, certification, statut indépendant (le cas échéant).
* Données techniques : adresse IP, logs de connexion, type d’appareil, navigateur.
* Données de configuration : contenu du Site Coach, images, textes, profils clients, offres.
* Données d’assistance : échanges avec le support.

## 2.2. Données collectées par Paddle

* Données de paiement et facturation.
* TVA, adresse de facturation, moyen de paiement.
  Ces données sont traitées selon la politique de Paddle.

## 2.3. Données collectées concernant les Clients Finaux

Données encodées par le Coach dans le CRM Ignite Coach :

* identité, coordonnées, historique de contact, notes de suivi, informations sportives ou physiques, objectifs.
  Ces données ne sont **pas** utilisées par M4 Entreprise à des fins propres.

# 3. FINALITÉS DU TRAITEMENT

## 3.1. Pour l’Utilisateur

* Fournir le Service SaaS Ignite Coach.
* Héberger et gérer le Site Coach.
* Gérer le compte, l’authentification et la sécurité.
* Assurer l’assistance technique et le support.
* Améliorer le Service (diagnostics, optimisation, mesure des performances).
* Respecter les obligations légales (facturation, comptabilité).

## 3.2. Pour les Clients Finaux

* Hébergement des données enregistrées par le Coach.
* Disponibilité de l’outil CRM et des fonctionnalités associées.
  M4 Entreprise agit uniquement sur instructions du Coach.

---

# 4. BASE LÉGALE DES TRAITEMENTS

* **Exécution du contrat** : création du compte, utilisation du Service, hébergement du Site Coach.
* **Obligations légales** : facturation via Paddle, sécurité, conservation minimale.
* **Intérêt légitime** : amélioration du Service, prévention des fraudes, analyses techniques.
* **Consentement** (si l’Utilisateur est consommateur ou pour certaines fonctionnalités facultatives).

---

# 5. DESTINATAIRES ET SOUS-TRAITANTS

Les données peuvent être transmises uniquement à :

* **Paddle** : paiements et factures.
* **Hébergeur cloud** localisé dans l’UE.
* Outils techniques (email transactionnel, CDN, monitoring), tous situés dans l’UE ou en pays reconnus adéquats.

Aucun transfert sans cadre légal valide (décision d’adéquation ou clauses contractuelles types).

Aucun partage commercial, aucune revente de données.

---

# 6. DURÉE DE CONSERVATION

* Données du Coach : conservées tant que le compte est actif, puis 90 jours maximum après résiliation (sauf obligation légale).
* Données des Clients Finaux : conservées tant que le Coach maintient son compte ou selon ses propres règles.
* Logs techniques : 12 mois maximum.
* Données de facturation (Paddle) : durée légale comptable.

* chiffrement des données en transit (HTTPS),
* firewalls et protections anti-intrusion,
* isolation des environnements (multi-tenant),
* sauvegardes régulières,
* contrôle d’accès strict selon les rôles,
* surveillance et audit technique.

L’Utilisateur doit utiliser des mots de passe sécurisés et protéger ses accès.
